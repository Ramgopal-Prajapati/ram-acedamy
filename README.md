# RamSir Academy Portal

A full-stack MERN application for managing academy operations with admin and student roles.

## Features

### Admin
- Student management (add, view students)
- Course management (create courses, assign to students)
- Assignment management (create weekly assignments)
- Fee management (track payments, view revenue)
- Assignment submission review and approval

### Student
- View assigned courses and details
- Submit assignments via GitHub URLs
- Track assignment submission status
- View fee payment summary
- Profile management

## Tech Stack

- **Frontend**: React + Vite, React Router DOM, React Icons, Pure CSS
- **Backend**: Node.js, Express.js
- **Database**: MongoDB with Mongoose
- **Authentication**: JWT-based
